# Landing-Page-Grão-de-Ouro
 Landing Page para empresa fictícia  responsável pela venda de café
 Landing page para uma marca fictícia de cafés, foco na utilização de CSS para uma UI bonita e bem estruturada, além do uso de javascript para programação de menus, pop up e um carrosel de imagens. Uso das cdn do google fonts e bootstrap icons para os icones. Além disso, a página foi idealizada a partir do mobile first.
